Create link on Post more conveniently.

== Description ==

When contributing a report by Wordpress, many external pages are described using the <a>tag, but it is troublesome. 
When introducing a link place, it contributes combining a title and URL in many cases, but the procedure is the following procedures. 

1. The page is opened, check the sauce of 
2. pages, copy the contents of the 
3. <title> tags, and insert in contribution.
 
In order to reduce this time-consuming work very much, when inputting URL, plug-in which keeps a title was created. 

== Installation ==

1. Install to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. If the [ez-link!] button is clicked from a post page, since the input dialog of URL will be displayed, please input URL according to liking. 
(However, it cannot be used in VisualEditor)
